## What is JumpServer? 
 
**JumpServer** is an open-source Privileged Access Management (PAM) tool that provides DevOps and IT teams with on-demand and secure access to SSH, RDP, K10s, Remote Apps and Database endpoints through a web browser.

![jumpserver-overview](/images/jumpserver-overview.png)

## Components

JumpServer consists of multiple key components, which collectively form the functional framework of JumpServer, providing users with comprehensive capabilities for operations management and security control.
